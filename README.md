# 🛍️ Customer Behavior Analysis

This project focuses on analyzing customer shopping behavior using **Python, SQL, and Power BI**. The goal is to uncover insights about purchasing patterns, customer preferences, and trends to support data-driven decision-making.

Project Overview

In this project, we:
- Cleaned and processed raw customer data
- Performed exploratory data analysis (EDA)
- Wrote SQL queries to extract meaningful insights
- Built interactive dashboards using Power BI
- Identified patterns in customer purchasing behavior

Tech Stack

- 🐍 Python (Pandas, NumPy)
- 🗄️ SQL (Data querying & transformation)
- 📊 Power BI (Dashboard & Visualization)
- 📓 Jupyter Notebook

Project Structure
├── customer_behavior_analysis.ipynb # Data analysis using Python
├── customer_shopping_behavior.sql # SQL queries for insights
├── custemer_behavior.pbix # Power BI dashboard
└── README.md # Project documentation


Power BI Dashboard

- Product performance analysis
- Interactive filters for deeper insights
